# 第II部　言語が提供するデータ構造の実装

第II部では、視点を反対側に移します。プログラミング言語は、利用者に対して
整数・文字列・配列・ハッシュ・オブジェクトといった「型」を提供します。
これらは利用者にとっては当たり前の道具ですが、処理系の内部では、
それぞれに工夫を凝らしたデータ構造として実装されています。

`1 + 2` がなぜ速いのか、巨大な整数がなぜ計算できるのか、
文字列の連結をどう効率化するのか、ハッシュはなぜ平均して一定時間で
引けるのか —— こうした「当たり前」の裏側を、実装の視点から解きほぐして
いきます。多くの例で、実在する言語処理系（とくに Ruby）の設計を
参照します。
